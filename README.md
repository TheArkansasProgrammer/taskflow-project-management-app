# TaskFlow

TaskFlow is a React-based project management dashboard that helps users organize tasks using a Kanban-style workflow.

## Features

- Task creation
- Task deletion
- Task status tracking
- Priority levels
- Due dates
- Dashboard metrics
- Local storage persistence
- Responsive design

## Technologies Used

- React
- Vite
- JavaScript
- CSS
- Git
- GitHub
